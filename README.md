# Funnel and Revenue Analysis in Ecommerce

## Summary
The goal of this project is to analyze user behaviour across an ecommerce funnel and identify key drivers of user conversion and revenue. The project focuses on the following:
- Funnel drop-off between stages
- Conversion perfromance by traffic source
- Revenue contribution by channel and product
- Time-to-conversion analysis
